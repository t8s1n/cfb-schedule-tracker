# CFB Schedule Calendars

Subscribe to these calendar URLs in Google Calendar:

- `cfb_schedule.ics` - All tracked games
- `cfb_maryland.ics` - Maryland games (example)

## How to Subscribe

1. Go to Google Calendar
2. Click the `+` next to "Other calendars"
3. Select "From URL"
4. Enter: `https://YOUR_USERNAME.github.io/cfb-schedule-tracker/cfb_schedule.ics`
5. Click "Add calendar"

The calendar will auto-update every 12-24 hours.
